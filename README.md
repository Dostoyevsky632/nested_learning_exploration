# Nested Learning Exploration

The purpose of this project is to explore the potential of the new **"nested learning"** paradigm introduced in the paper *Nested Learning: The Illusion of Deep Learning Architecture*.

This repository will be used to organize and package the relevant experimental code, implementation notes, and experimental records related to our reproduction and exploration efforts.

## Project Goals

- Investigate the core ideas of nested learning from the paper
- Build experimental prototypes based on the proposed architecture and learning dynamics
- Compare against suitable baselines to evaluate the practical value of the paradigm
- Record experimental settings, results, and observations in a reproducible way

## Repository Contents

The experimental code will be uploaded to this repository together with:

- model implementations
- training and evaluation scripts
- baseline comparisons
- experiment logs and records
- notes on design choices and findings

## Current Status

This repository is currently being prepared as an experimental workspace for nested learning research.
